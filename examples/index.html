Welcome To My Gulp Project

With pure es5, using gulp will auto order can concatenating your files base off of their dependencies. If you have circular dependencies, then you'll need a more advance setup.

To run gulp, just do

gulp
//or
gulp default

As for making one bundle.js to be run in a browser, you'll want to use browserify. You almost dont need gulp if you wanted to as you can just do 

browserify index.js > bundle.js
